## Условие задачи  

Составить программу, которая во введенной пользователем
строке вставляет восклицательный знак после второго слова предложения

## Алгоритм

начало  
программа запрашивает у пользователя предложение. Строка сохраняется в переменную `text`.  
создаём пустую строку `res` для результата, счётчик слов `count = 0`.  
вызываем `strtok(text, " ")` для получения первого слова в `part`.  
пока есть слова (`part != NULL`):  
- увеличиваем `count`  
- добавляем слово `part` в `res`  
- если `count == 2`: добавляем восклицательный знак `!` в `res`  
- получаем следующее слово `strtok(NULL, " ")`  
- если следующее слово есть: добавляем пробел в `res`  
выводим результат: печатаем строку `res` на экран.  
Конец
